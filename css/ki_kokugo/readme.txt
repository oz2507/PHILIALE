このフォントは、以下のフォントにオリジナルの文字を加え加工したものです。
「M+（エムプラス）」フォント
「源ノ角ゴシック」フォント

【ライセンス】
このフォントは、「M+」フォント、「源ノ角ゴシック」フォントにオリジナルの文字を加え加工したものです。
このフォントのライセンスは、「M+」フォントと「源ノ角ゴシック」フォントのライセンスに準じます。
「M+」フォントのライセンスは、配布物に含まれる「mplus-TESTFLIGHT-057」フォルダ内の書類をご覧ください。
「源ノ角ゴシック」フォントのライセンスは、配布物に含まれる「Apache License 2.0.txt」をご覧ください。
ひらがな・全角カタカナ部分のデザインは「フリーフォントの樹」によるものです。

【源ノ角ゴシックからの変更点】
・OpenType形式をTrueType形式に変更
・M+フォントが収録している文字をすべてそれに置き換え（不足文字のみ源ノ角ゴシックを使用するため）
・仮名文字をオリジナルとし、フォント名を「刻ゴシック」とする
・行間等を独自の設定に置き換える

【商用利用・プログラムへの埋め込みについて】
作成した印刷物およびデジタル・コンテンツにつき、その商用・非商用にかかわらず
放送、通信、各種記録メディアなどの媒体の形式も問わず、使用をすることができます。
ただし、これは「M+」フォントと「源ノ角ゴシック」フォントのライセンスのライセンスのもとにあります。
